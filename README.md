# Healtpredec

![Project image](demo/front_page.gif)

Healtpredec provides many services which help to predict the various disease like (malaria, diabetes, cancer, heart disease , stroke, etc)

### Various models used

* Breast Cancer model 
* Diabetes Model
* Pneumonia model
* Stroke model
* Maleria model
* Liver disease model
* Heart disease model
